# Falppy Bird

Small Flappy Bird-style game (HTML + Canvas + JavaScript). No dependencies or assets required.

## How to run

1. Open `index.html` in a browser (Chrome/Edge recommended).
2. Use `Click / Tap` or press `Space` to flap.
3. When you hit a pipe or the ground, press `Restart` (or `Space`) to play again.

## Files

- `index.html` - page layout
- `styles.css` - styling
- `game.js` - game logic
